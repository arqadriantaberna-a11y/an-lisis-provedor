# Control de Proveedores

Página web simple para cargar proveedores, empleados y evaluaciones con observaciones.

## Cómo usar en GitHub Pages

1. Crear un repositorio nuevo.
2. Subir el archivo `index.html`.
3. Ir a Settings > Pages.
4. En Branch elegir `main` y carpeta `/root`.
5. Guardar.
6. GitHub te dará una URL pública.

Los datos se guardan en el navegador con LocalStorage.
Usá el botón Backup para no perder información.
